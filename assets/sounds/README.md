# Sons do EduQuest

Gere os sons usando:
- https://freesound.org/ (busque por "success", "error", "click")
- Ou use o Web Audio API para gerar sons programaticamente
- Ou grave sons simples com seu celular

Arquivos necessários:
- correct.wav (som de acerto - 0.5s)
- wrong.wav (som de erro - 0.5s)
- complete.mp3 (som de conclusão - 2s)
- click.wav (som de clique - 0.1s)
