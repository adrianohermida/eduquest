## 🔄 2. ALGORITMO DE REVISÃO ESPAÇADA: Configuração Técnica

### Lógica do Sistema de Repetição
```
Estado Inicial: Questão respondida corretamente pela 1ª vez
↓
Intervalo 1: 1 hora → Reaparece em flashcard rápido
↓
Se acerto: Intervalo 2: 24 horas
↓
Se acerto: Intervalo 3: 3 dias
↓
Se acerto: Intervalo 4: 7 dias
↓
Se acerto: Intervalo 5: 15 dias → Questão "consolidada"
↓
Se erro em qualquer etapa: Reset para Intervalo 1 + micro-aula de 30s
```

### Configuração no Backend (Pseudocódigo)
```python
class SpacedRepetition:
    INTERVALS = [3600, 86400, 259200, 604800, 1296000]  # em segundos
    
    def schedule_next_review(self, user_id, question_id, is_correct):
        current_stage = get_user_stage(user_id, question_id)
        
        if is_correct:
            if current_stage < len(self.INTERVALS) - 1:
                next_stage = current_stage + 1
                next_review = now() + self.INTERVALS[next_stage]
                update_user_stage(user_id, question_id, next_stage, next_review)
            else:
                mark_as_consolidated(user_id, question_id)
        else:
            # Erro: reset + intervenção pedagógica
            reset_to_stage_1(user_id, question_id)
            trigger_micro_lesson(user_id, question_id)
            schedule_review(user_id, question_id, now() + self.INTERVALS[0])
```

### Integração com Flashcards
- **Flashcards Nível 1**: Aparecem no "Modo Treino Diário" com prioridade baseada no intervalo
- **Fila inteligente**: Questões com erro recente têm prioridade na fila do dia
- **Modo "Revisão Relâmpago"**: 5 flashcards aleatórios de estágios 3-5 para manutenção de longo prazo

### Dashboard do Aluno: Progresso de Consolidação

## ❤️ 3. GATILHOS EMOCIONAIS/SENSORIAIS: Implementação de "Dicas" Multimodais

### Sistema de Micro-Áudios (2-3 segundos)
| Tipo de Dica | Áudio Exemplo | Gatilho Emocional | Ícone Associado |
|-------------|--------------|------------------|-----------------|
| **Memória Visual** | *Som de "clique" suave + sussurro "Absoluta!"* | Curiosidade + Confirmação | 👁️ |
| **Associação Sonora** | *Som de ônibus + buzina leve* para migração pendular | Rotina + Identificação | 🚌 |
| **Reforço Positivo** | *Som de "brilho" ou "sininho"* ao acertar | Serotonina + Conquista | ✨ |
| **Alerta Pedagógico** | *Som suave de "atenção"* ao errar | Foco + Correção sem frustração | 💡 |

### Especificações Técnicas dos Áudios
```json
{
  "audio_specs": {
    "duration": "2-3 seconds",
    "format": "MP3, 44.1kHz, 128kbps",
    "volume_normalization": true,
    "accessibility": {
      "visual_alternative": true,
      "transcript_available": true,
      "volume_control": "user_adjustable"
    }
  }
}
```

### Pop-ups Visuais com Ícones Consistentes
- **Biblioteca de Ícones**: Criar conjunto de 50 ícones vetoriais (SVG) para todos os tópicos
- **Animação sutil**: Fade-in de 200ms, permanência de 3s, fade-out de 200ms
- **Posicionamento**: Canto inferior direito, não obstruindo o conteúdo principal
- **Acessibilidade**: Ícones com alt-text e opção de desativar para usuários com sensibilidade visual

### Exemplo de Implementação Frontend (React)
```jsx
const TipTrigger = ({ tip, audioFile, icon }) => {
  const [showTip, setShowTip] = useState(false);
  
  const playAudio = () => {
    const audio = new Audio(`/audio/tips/${audioFile}`);
    audio.volume = 0.3; // Volume suave
    audio.play();
  };
  
  return (
    <button 
      className="tip-trigger"
      onMouseEnter={() => { setShowTip(true); playAudio(); }}
      onMouseLeave={() => setShowTip(false)}
      aria-label={`Dica: ${tip}`}
    >
      <Icon name={icon} size="sm" />
      {showTip && (
        <div className="tip-popup animate-fade">
          <p className="tip-text">{tip}</p>
          <span className="tip-icon">{icon}</span>
        </div>
      )}
    </button>
  );
};
```
